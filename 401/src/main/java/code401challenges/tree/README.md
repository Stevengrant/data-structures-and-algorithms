# Trees
Write a tree class as well as a binary search tree
## Challenge
#### preOrder

###### Returns array in preorder.
* Time: O(n)
* Space O(n)

#### inOrder

###### Returns array in inOrder.
* Time: O(n)
* Space O(n)

#### postOrder

###### Returns array in postOrder.
* Time: O(n)
* Space O(n)

## Binary Search Tree
#### add
* Time: O(N log N)

#### contains
* Time: O(n log n)
* Space O(1)
